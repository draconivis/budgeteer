# Budgeteer

A simple webapp to track finances, tailored to my need

## Requirements

### MVP
- User can set a budget
- User can track expenses / reimbursements

### Further
- User can add tags to expenses / reimbursements
- historical data
- data visualization

## Technology
- Symfony (duh)
    - Doctrine + SQLite
- FrankenPHP
